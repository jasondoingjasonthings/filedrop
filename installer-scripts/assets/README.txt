Place a 256x256 .ico file named "filedrop.ico" in this folder before compiling the installers.

If you don't have an icon, remove the SetupIconFile line from both .iss files and Inno Setup
will use its default icon instead.

Free icon tools:
  - https://www.icoconverter.com  (convert PNG to ICO)
  - https://realfavicongenerator.net (generate from any image)
